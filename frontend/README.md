# My Todo App

A collaborative todo app built with SolidJS where users can manage their own todos and browse others'.

## Core Features

- [ ] Users can view other people's todos (read-only)
- [ ] Users can create their own todos
- [ ] Users can view their own todos on a dedicated page
- [ ] Users can edit their own todos
- [ ] Users can delete their own todos
- [ ] Users can update their own todos
- [ ] Users **cannot** edit, delete, or update other users' todos

## Usage

```bash
npm install # or pnpm install or yarn install
```

Learn more on the [Solid Website](https://solidjs.com) and come chat with us on our [Discord](https://discord.com/invite/solidjs).

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in the development mode.  
Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

### `npm run build`

Builds the app for production to the `dist` folder.  
It correctly bundles Solid in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.  
Your app is ready to be deployed!
